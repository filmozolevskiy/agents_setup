## How to use on cards

- **Dev work** (code changes, fixes, new integrations) → developer.
- **Test coverage / regression / verification** → QA.
- **Prevalence numbers, debug-log queries, scoping, metrics** → Analyst.
- **Ancillaries (bags, seats, services)** → Lauren.
- **Post-ticketing (refunds, exchanges, voids, schedule changes)** → Viorel.
- **Manual agent workflows / escalations / operational impact** → Emerlie.

When unsure, mention the area lead in a comment rather than assigning the card, and ask the user who to route it to.
